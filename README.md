##Building
Currently building is only tested on Ubuntu 16.04.

1. Install MonoDevelop 5.10 and MonoGame 3.5
2. Clone repository
3. Add project to MonoDevelop
4. Build Content Pipeline file Content/Content.mgcb
5. Build and run project
